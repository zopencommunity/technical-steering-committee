## ‚öôÔ∏è z/OS-Critical Tooling Improvements

We invested heavily in improving support for **preserving extended attributes and file tags**, which are critical on z/OS.

### Enhanced Tools
* `coreutils` (`mv`, `cp` preserve extended attributes with the `-p` option)
* `tar` - now preserves file tags and extended attributes
* `rpm` - now preserves file tags and extended attributes
* `rsync` - now preserves file tags when syncing from z/OS to z/OS

### üåê Curl Auto-Tagging
We added auto-tagging capabilities to **curl**:
1.  **Auto-Tag:** When you download, it will automatically tag the file based on the file content
2.  **Environment:** It continues to honour `_ENCODE_FILE_NEW`.
3.  **Override:** You can override the tag manually using the new flag: `--tag <ccsid>`.
